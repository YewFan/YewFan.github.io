3/2/2023: 
个人感觉现在弱人工智能随著chatgpt的爆红已经得到了社会的认可，同时我相信社会也在期待着人工智能可以向下一个阶段前进。人工智能的目的是实现对人类智能的模仿。逻辑在人工智能的应用是必不可少的，说到底人脑的做出对简单事物的理性判断，基本上源于归纳，演绎等一些经典逻辑。但是面对复杂事物时，使用经典逻辑并非是一个好的选择。所以非经典逻辑如模态逻辑，模糊逻辑，动态逻辑等对复杂事物的判断更加准确，就像人在面对未知事物时做出的判断应该以准不准确为主，而不是正确与否，这里不考虑知识范畴。于是我相信非经典逻辑中一些逻辑一定会在人工智能中得到广泛应用，虽然事实也是如此。所以我将学习逻辑纳入我的学习规划中。数学，计算机也都是人工智能的基础。
